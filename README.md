# Plus Ultra

**Plus Ultra** is a CSS library designed to extend and empower **Bulma CSS**, transforming it with an aesthetic inspired by **Material Design**.

This project aims to combine the simplicity and modularity of Bulma with the elegance and interactions typical of Material Design, offering a set of modern and responsive components.

## Installation

Use your favorite package manager to add Plus Ultra to your project:

```bash
pnpm add plus-ultra
# or
npm install plus-ultra
# or
yarn add plus-ultra
```

## Usage

Import the main SCSS file into your project:

```scss
@use "plus-ultra";
```
