# Table

The inevitable HTML table, with special styles.

## Basic Example

```html
<table class="table is-striped is-hoverable is-fullwidth">
  <thead>
    <tr>
      <th>Pos</th>
      <th>Team</th>
      <!-- ... -->
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>1</th>
      <td>Leicester City</td>
      <!-- ... -->
    </tr>
  </tbody>
</table>
```
