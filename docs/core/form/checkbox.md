# Checkbox

The 2-state checkbox in its native format.

## Basic Example

```html
<label class="checkbox">
  <input type="checkbox" />
  Remember me
</label>
```

## Checked and Disabled

```html
<label class="checkbox">
  <input type="checkbox" checked />
  Checked
</label>
<label class="checkbox">
  <input type="checkbox" disabled />
  Disabled
</label>
```
