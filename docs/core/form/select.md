# Select

Select dropdowns.

## Basic Example

```html
<div class="field">
  <div class="control">
    <div class="select">
      <select>
        <option>Select dropdown</option>
        <option>With options</option>
      </select>
    </div>
  </div>
</div>
```

## Material Filled

```html
<div class="field">
  <div class="control">
    <div class="select is-material-filled">
      <select>
        <option>Filled Select</option>
        <option>With options</option>
      </select>
    </div>
  </div>
</div>
```
